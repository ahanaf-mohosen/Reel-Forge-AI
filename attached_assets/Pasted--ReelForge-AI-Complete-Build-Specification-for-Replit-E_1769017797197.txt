# ReelForge AI - Complete Build Specification for Replit

## Executive Summary
Build a production-ready AI-powered web application that automatically converts long-form videos into viral short-form reels (9:16 vertical format) optimized for Instagram Reels, TikTok, and YouTube Shorts.

---

## 🎯 Product Overview

### Core Value Proposition
Users upload a video file, paste a YouTube link, or provide a public video URL. The AI analyzes the content, identifies the most engaging moments, and generates 3 ready-to-publish vertical reels (20-40 seconds each).

### Target Users
- Content creators
- Social media managers
- Marketing teams
- Influencers
- Small businesses

---

## 🏗️ Technical Architecture

### Technology Stack

**Frontend:**
- React 18+
- TailwindCSS for styling
- Framer Motion for animations
- Lucide React for icons
- Recharts for analytics (optional)
- React Router for navigation

**Backend:**
- Node.js (v18+)
- Express.js
- Multer for file uploads
- FFmpeg for video processing
- Fluent-FFmpeg wrapper
- OpenAI API (GPT-4 for analysis)
- Whisper API for transcription
- ytdl-core for YouTube downloads

**Storage:**
- Local filesystem (uploads/, outputs/)
- JSON-based project database (projects.json)
- Extensible to AWS S3/Supabase

**External Dependencies:**
- OpenAI API (GPT-4 + Whisper)
- FFmpeg binary
- Node.js libraries

## 🤖 AI Processing Pipeline

### Detailed Workflow

**Step 1: Video Input**
- Accept: MP4, MOV, AVI (up to 500MB)
- Validate file type and size
- Generate unique project ID
- Save to `uploads/[projectId]/original.[ext]`

**Step 2: Audio Extraction**
```javascript
// Using FFmpeg
ffmpeg -i input.mp4 -vn -acodec pcm_s16le -ar 16000 -ac 1 audio.wav
```

**Step 3: Transcription**
```javascript
// Using OpenAI Whisper API
const transcription = await openai.audio.transcriptions.create({
  file: fs.createReadStream('audio.wav'),
  model: 'whisper-1',
  language: 'en',
  response_format: 'verbose_json',
  timestamp_granularities: ['segment']
});
```

**Step 4: Highlight Detection**
```javascript
// GPT-4 prompt
const prompt = `You are an expert viral video editor specializing in short-form content for TikTok, Instagram Reels, and YouTube Shorts.

Analyze the following video transcript and identify the 3 BEST moments for creating engaging short-form reels.

CRITERIA:
- Each clip must be 20-40 seconds long
- Look for moments with:
  * High emotional impact
  * Clear hooks or payoffs
  * Self-contained stories or ideas
  * Visual or verbal punchlines
  * Surprising or valuable information
- Avoid clips that start mid-sentence or lack context
- Prioritize content that works standalone

TRANSCRIPT:
${transcription.text}

TIMESTAMP SEGMENTS:
${JSON.stringify(transcription.segments)}

Return ONLY valid JSON in this exact format:
{
  "clips": [
    {
      "start": 12.5,
      "end": 45.3,
      "reason": "Strong hook about unexpected benefit",
      "suggestedCaption": "This changed everything 🔥"
    }
  ]
}`;

const response = await openai.chat.completions.create({
  model: 'gpt-4',
  messages: [{ role: 'user', content: prompt }],
  response_format: { type: 'json_object' }
});
```

**Step 5: Video Cutting**
```javascript
// For each clip
ffmpeg -i original.mp4 -ss [start] -t [duration] -c copy clip_temp.mp4
```

**Step 6: Vertical Formatting**
```javascript
// Resize to 1080x1920 (9:16) with smart cropping
ffmpeg -i clip_temp.mp4 \
  -vf "scale=1080:1920:force_original_aspect_ratio=increase,crop=1080:1920" \
  -c:a copy \
  reel_1.mp4
```

**Step 7: Optional Enhancements**
- Add subtle zoom effect for engagement
- Normalize audio levels
- Add fade in/out
- Generate auto-captions using Whisper

**Step 8: Save & Return**
- Save to `outputs/[projectId]/reel_[n].mp4`
- Update project database
- Return URLs and metadata

## 🔌 API Specification

### Base URL
`http://localhost:5000/api`

### Endpoints

**POST /upload**
- **Description:** Upload video file or URL
- **Content-Type:** multipart/form-data or application/json
- **Request Body (file):**
  ```javascript
  {
    file: File, // via FormData
    projectName: string (optional)
  }
  ```
- **Request Body (URL):**
  ```javascript
  {
    url: string,
    type: 'youtube' | 'direct',
    projectName: string (optional)
  }
  ```
- **Response:**
  ```javascript
  {
    success: true,
    projectId: string,
    videoPath: string,
    metadata: {
      filename: string,
      size: number,
      duration: number,
      format: string
    }
  }
  ```

**POST /process**
- **Description:** Start AI processing
- **Request Body:**
  ```javascript
  {
    projectId: string,
    options: {
      clipCount: number (default: 3),
      minDuration: number (default: 20),
      maxDuration: number (default: 40)
    }
  }
  ```
- **Response:**
  ```javascript
  {
    success: true,
    projectId: string,
    status: 'processing'
  }
  ```

**GET /process/:projectId/status**
- **Description:** Get processing status
- **Response:**
  ```javascript
  {
    projectId: string,
    status: 'uploading' | 'transcribing' | 'analyzing' | 'cutting' | 'formatting' | 'completed' | 'failed',
    progress: number (0-100),
    currentStep: string,
    estimatedTimeRemaining: number (seconds),
    error: string (if failed)
  }
  ```

**GET /projects**
- **Description:** Get all projects
- **Query Params:**
  - page: number
  - limit: number
  - status: string
  - sort: 'newest' | 'oldest'
- **Response:**
  ```javascript
  {
    projects: [
      {
        id: string,
        name: string,
        createdAt: timestamp,
        status: string,
        thumbnail: string,
        reelCount: number,
        originalVideo: {
          filename: string,
          duration: number,
          size: number
        }
      }
    ],
    total: number,
    page: number,
    pages: number
  }
  ```

**GET /projects/:projectId**
- **Description:** Get project details
- **Response:**
  ```javascript
  {
    id: string,
    name: string,
    createdAt: timestamp,
    status: string,
    originalVideo: { ... },
    reels: [
      {
        id: string,
        url: string,
        thumbnail: string,
        duration: number,
        caption: string,
        start: number,
        end: number
      }
    ],
    transcript: string
  }
  ```

**GET /reels/:projectId/:reelId/download**
- **Description:** Download specific reel
- **Query Params:**
  - format: 'mp4' | 'mov'
  - quality: '720p' | '1080p'
- **Response:** Video file stream

**POST /edit**
- **Description:** Edit reel (trim, add captions)
- **Request Body:**
  ```javascript
  {
    projectId: string,
    reelId: string,
    edits: {
      trim: {
        start: number,
        end: number
      },
      caption: {
        text: string,
        font: string,
        size: number,
        color: string,
        position: 'top' | 'center' | 'bottom'
      },
      audio: {
        volume: number (0-100),
        fadeIn: boolean,
        fadeOut: boolean
      }
    }
  }
  ```
- **Response:**
  ```javascript
  {
    success: true,
    reelUrl: string
  }
  ```

**DELETE /projects/:projectId**
- **Description:** Delete project and all reels
- **Response:**
  ```javascript
  {
    success: true,
    deletedProjectId: string
  }
  ```

## 🔒 Security & Validation

### File Upload Validation

```javascript
const ALLOWED_TYPES = ['video/mp4', 'video/quicktime', 'video/x-msvideo'];
const MAX_FILE_SIZE = 500 * 1024 * 1024; // 500MB

function validateUpload(file) {
  if (!ALLOWED_TYPES.includes(file.mimetype)) {
    throw new Error('Invalid file type. Only MP4, MOV, and AVI are allowed.');
  }
  
  if (file.size > MAX_FILE_SIZE) {
    throw new Error('File too large. Maximum size is 500MB.');
  }
  
  return true;
}
```

### Environment Variables

```bash
# .env
NODE_ENV=development
PORT=5000
OPENAI_API_KEY=sk-...
MAX_UPLOAD_SIZE=524288000
ALLOWED_ORIGINS=http://localhost:3000
```

### API Key Protection

- Never expose API keys in client-side code
- Store in `.env` file (never commit)
- Access via `process.env.OPENAI_API_KEY`
- Validate on server startup

### Rate Limiting

```javascript
const rateLimit = require('express-rate-limit');

const uploadLimiter = rateLimit({
  windowMs: 15 * 60 * 1000, // 15 minutes
  max: 5, // 5 uploads per window
  message: 'Too many uploads. Please try again later.'
});

app.post('/api/upload', uploadLimiter, uploadHandler);
```
